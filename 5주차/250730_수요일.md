## 깃/깃허브 특강

- 깃을 통해서 버전관리를 하고 원하는 버전으로 돌아갈 수 있다
- 정적 페이지 호스팅 가능
- GUI 툴로 SourceTree 가장 많이 사용
- 커밋 컨벤션
- https://www.tablesgenerator.com/markdown_tables 마크다운 테이블
- feature/기능명으로 브랜치를 열어서 기능 작업
- 나중에 main 브랜치로 병합
- PR(Pull Request)를 통해서 동의를 얻어서 병합
